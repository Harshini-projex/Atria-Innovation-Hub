Atria Innovation Hub

A responsive web application that provides a platform for Atria Institute of Technology students to explore, showcase, and share innovative projects.

📌 Overview

Atria Innovation Hub is a frontend web application developed using React.js. It is designed to promote innovation among students by providing a clean and user-friendly interface where projects can be explored, searched, and showcased.

The application focuses on delivering a modern and responsive user experience and serves as the frontend prototype of a larger innovation platform.

✨ Features
🔐 Student Login Interface
🏠 Dashboard
🔍 Search Bar
📂 Project Cards
➕ Add Project Button (UI)
❤️ Like Button (UI)
⭐ Rating Interface (UI)
💬 Comment Section (UI)
📝 My Reviews Section
🚪 Logout Button
📱 Fully Responsive Design

Note: Backend functionality such as authentication, database storage, and project management is not yet implemented. The current project focuses on frontend development.

🛠️ Tech Stack
React.js
JavaScript (ES6)
HTML5
CSS3

🚀 Getting Started
Clone the Repository
git clone https://github.com/your-username/Atria-Innovation-Hub.git
Navigate to the Project
cd Atria-Innovation-Hub
Install Dependencies
npm install
Run the Application
npm start

The application will open in your browser at:

http://localhost:3000
